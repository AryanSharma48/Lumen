# Economics of Lumen

### Unit Economics
*   **Customer Lifetime Value (LTV):** $1,000 (Average success fee per converted startup).
*   **Customer Acquisition Cost (CAC):** Target < $200 (Scrappy distribution, university channels, organic sharing).
*   **LTV/CAC Ratio:** 5x (Target).

Justification: If Lumen identifies $500/mo ($6,000/yr) in wasted spend, Credex negotiates the new contracts and takes a standard 15-20% broker margin, netting roughly $1,000 per converted startup.

### Conversion Funnel
1.  **Free Audits:** 1,000 units
2.  **Consultation Booked (10%):** 100 calls
3.  **Closed Won (20% of calls):** 20 buyers
4.  **Total Conversion Rate:** 2% (Audit → Buyer)

### Path to $1M ARR (18 Month Horizon)
To reach $1,000,000 in Annual Recurring Revenue (or total contract value per year):

*   **Total Buyers Needed:** $1,000,000 / $1,000 LTV = 1,000 buyers.
*   **Total Audits Needed:** 1,000 / 0.02 (Conversion Rate) = 50,000 audits.
*   **Timeline:** 18 months = 547 days.
*   **Daily Audit Target:** 50,000 / 547 ≈ **92 audits / day.**
